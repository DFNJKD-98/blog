好物推荐

发表于2022-07-07|更新于2022-07-07
|阅读量:

一、设计篇

  • nodesign 各种设计门类的汇总网站
  • palettable 各种颜色色号推荐
  • unicode 图标设计
  • wallhaven 高清壁纸
  • canva 在线平面设计网站
  • 开发者藏经阁 阿里系开发者电子书合集
  • learnJS 微软出的简单js入门视频

二、参考文档

  • mdn web标准
  • dash 一个阅读各种官方文档的app，目前 Mac、ios 都可以下载

三、画图

  • excalidraw 非常好用的手绘风画图网站，性能不错易上手，而且开源，既适合使用也适合学习

三、辅助学习

  • CSS

    • cssbattle 一个在线css对战平台，给一个设计稿你来还原，最终系统给一个评分，可以看到你和大佬的差距
    • learn layout 特别轻便的css布局学习网站

  • JavaScript

    • jstips 短小的js小技巧分享网站
    • caniuse 一个可视化的浏览器兼容性查询网站
    • stateofjs JavaScript趋势可视化，可以看到各大技术、框架当前的热度、使用情况
    • jsperf js性能测试

  • refactoringguru 学习代码重构和设计模式

  • regex 在线可视化正则表达式测试网站

  • spectrum 一个英文的大型社区网站，可以和全世界的开发者交流

  • youtube 播主：JSConf

  • geektime 提取码：lkzk
